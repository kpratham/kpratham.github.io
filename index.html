Cross-Age Face Recognition using DL

Seoul, S. Korea

CVPR 2019

Problem Statement : All the current systems can be used for recognizing people only at same age. We developed a system to keep an eye on human trafficking.

Proposed Solution : Facenet

Facenet (Tensorflow implementation of OpenFace) was used to identify and recognize people in a video frame. All the current systems can be used for recognizing people only at same age. We developed a system to keep an eye on human trafficking. By fine-tuning the current available Facenet code with Large-Age-Gap dataset we tried to match missing people in a cctv footage.
